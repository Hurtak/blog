articles directory structure

0001-first-article/
    first four digits '0001' will be used as id in database (with leading zeroes
    removed)
    text after fist dash 'first-article' will be used as url

0001-first-article/article.html
    article's content in pure html

0001-first-article/data.json
    data inseted into database:
        "title"
        "preview"
        "publication_date"
        "last_update"
        "visible"

0001-first-article/image.png
    article preview image

0001-first-article/img/
    directory for articles images
